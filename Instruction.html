Step-by-step instructions on how to set up Git and GitHub

Git Set-up

Take the following steps to set up Git on your computer:

  1. Open your browser and go to the official website of Git: https://git-scm.com
  2. Go to download, then Select which type of operating system(os) you are using.
  3. Click the "click here to download" to download the latest vesion of Git(2.51.2)x64
  4. Once downloaded, click install
  5. To check if Git is now installed, go to an empty folder then right-click. If you see the Git bash, then it is installed.

VSCode Set-up

Take the following steps to set up VS code on your computer:

  1. Open your browser and go to the official website of VS code: https://code.visualstudio.com
  2. Go to download, then Select which type of operating system(os) you are using.
  3. Click the "click here to download" to download the latest vesion of VS code
  4. Once downloaded, click install
  5. To check if VS code is now installed, go to files and type in the address bar "cmd" then enter.
  6. In the command prompt, type "vs code" then enter. It will open VS code.
  7. Once VS Code open, you can adjust its settings of your choice (auto save, spacing and so forth)

GitHub repository Set-up

Take the following steps to set up your repository on your GitHub

  1. Open your browser and go to the official website of GitHub: https://github.com
  2. Log in your account or Sign up if you don't have an account.
  3. Once logged in, click the "+" icon on the top right corner of the page, then select "New repository".
  4. Fill in the repository name and description (optional), then set to public.
  5. Click "Create repository" to finish setting up your GitHub repository.

Connecting Git to GitHub repository including essential commands

Take the following steps to connect you Git to your created GitHub repository:

  1. In VS Code, open Git Bash in the folder where you want to connect to your GitHub repository.
  2. Open your browser and go to https://codemy.com/git/ and copy the commands to Initialize Git.
    Enter these one by one to your Git bash terminal
    $ git init
    $ git config --global user.name "Your Name"
    $ git config --global user.email "you@youraddress.com"
    $ git config --global push.default matching
    $ git config --global alias.co checkout
  3. Next, copy the commands to connect to your GitHub repository from the same page.
    Enter these one by one to your Git bash terminal
    $ git init
    $ git add README.md
    $ git commit -m "first commit"
    $ git branch -M main
    $ git remote add origin (your repository URL)
    $ git push -u origin main
  4. Your local Git is now connected to your GitHub repository.
